### INSTALL
#### 0. Install Node.js and Download zip:

Click install
http://nodejs.org/

After that

```sh
$ node -v
$ cd gulp_template
$ npm init
```

#### 1. Install gulp globally:

```sh
$ npm install --global gulp
```

#### 2. Install gulp in your project devDependencies:

```sh
$ npm install --save-dev gulp
```

#### 3. Run gulp

```sh
$ gulp
```
